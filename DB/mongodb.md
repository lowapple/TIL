# Install

```
# Mac homebrew user
brew install mongodb

# Window scoop user
scoop install mongodb
```

# Usage

### On Server
```
sudo mongod
```

### Mongodb Terminal
```
mongo
```

```
show dbs
```
```
use -db
```
```
show collections
```
```
collection.find()
collection.find({})
collection.find({}).count()
```
